│ RUTAS Y ENDPOINTS                                                            │
└──────────────────────────────────────────────────────────────────────────────┘

🌐 25+ RUTAS DEFINIDAS
  📖 Públicas:
    GET  /                          → Inicio
    GET  /categories                → Lista de categorías
    GET  /categories/{slug}         → Detalle de categoría
    GET  /threads/{slug}            → Detalle de hilo
    GET  /users/{id}                → Perfil de usuario

  🔐 Autenticadas:
    GET  /threads/create            → Formulario crear hilo
    POST /threads                   → Guardar hilo
    POST /threads/{id}/posts        → Crear respuesta
    PUT  /threads/{id}              → Actualizar hilo
    DELETE /threads/{id}            → Eliminar hilo

  👮 Moderación:
    POST /threads/{id}/lock         → Bloquear/desbloquear
    POST /threads/{id}/pin          → Fijar/desfijar
    POST /posts/{id}/solution       → Marcar solución
